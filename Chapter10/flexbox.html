Explantion of the topics:

  1. Flexbox Introduction:

    • Flexbox (Flexible Box Layout) is a CSS layout model that simplifies creating complex layouts.
    • It provides a flexible way to align elements and distribute space within a container element.
    • Note: Flexbox layout is most appropriate to the components of an application, and small-scale layouts, while the Grid layout is intended for larger scale layouts.

  2. Terminologies for flexbox:

    • Flex-Container or Flex-Parent:   The HTML element that has "display: flex" set on it is called the flex-container or flex-parent.
    • Flex-Item or Flex-Child:   The items(HTML elements) inside the flex container are called flex items.
    • Main-axis:
      • The main axis of a flex container is the primary axis along which flex items are laid out.
      • The main axis is either horizontal or vertical depending on the value of the flex-direction.
      • For flex-direction value row & row-reverse the main axis is horizontal.
      • For flex-direction value column & column-reverse the main axis is vertical.
    • Cross-axis: The axis perpendicular to the main axis is called the cross axis. Its direction depends on the main axis direction.

  3.  display: flex;

    • display: flex; make an element a flex container. It enables all its direct children for flexbox properties.
    • All the direct children (flex-items) of the flex container arranged according to the flex layout, their default inline or block nature does not work.
    • display: flex; This makes the flex container behave like a block-level element. The flex-container takes up the entire available width of its parent element.
    • display: inline-flex; This makes the flex-container behave like an inline-level element. This allows other inline elements to flow alongside it.

  4. Flexbox Properties:

    • Flex Container Properties: (We apply these properties on the flex container, and not on its items.)
      • flex-direction
      • flex-wrap
      • flex-flow
      • justify-content
      • align-items
      • align-content
      • gap, column-gap, row-gap
    • Flex Item Properties: (We apply these properties to individual flex items.)
      • order
      • align-self
      • flex-grow
      • flex-shrink
      • flex-basis
      • flex

  5. flex-direction property (flex-container property):

    • The flex-direction CSS property sets how flex items are placed in the flex container defining the main axis and the direction (normal or reversed).
    • Values can be:
      • row: This displays the flex-items horizontally from left to right. (default value)
      • row-reverse: It displays the flex items from right to left.
      • column: This displays the flex-items vertically from top to bottom.
      • column-reverse: It displays the flex items from the bottom to the top.
    • Syntax: 
          
        .container {
            display: flex;
            flex-direction: row-reverse; 
        }
    • Demo: Go


  6. flex-wrap property (flex-container property):

    • The flex-wrap CSS property sets whether flex items are forced onto one line or can wrap onto multiple lines.
    • Values can be:
      • nowrap: This keeps all the flex items on a single line either in a row or column. It allows the flex items to overflow if there's not enough room in the flex container. (default value)
      • wrap: This will wrap or push the flex items to the next line if there's not enough room for them.
      • wrap-reverse: Flex items will wrap onto multiple lines from bottom to top.
    • Syntax: 
          
        .container {
            display: flex;
            flex-wrap: wrap; 
        }
    • Demo: Go


  7. flex-flow property:(flex-container property)

    • The flex-flow property is a shorthand property for flex-direction & flex-wrap
    • Values can be:
      • Single value: We can any 1 value from flex-direction & flex-wrap property value.
      • Double value: 1 value from flex-direction property value & another 1 from flex-wrap property value, seperated by space. Ordering does not matter.
    • Syntax: 
          
        .container {
            display: flex;
            flex-flow: row-reverse wrap;
        }
    • Demo: Go


  8. justify-content property:(flex-container property)

    • The justify-content CSS property defines how the browser distributes extra space between and around content items along the main axis.
    • Values can be:
      • flex-start: This will place the flex-items at the start of the flex-direction (relative to flex-direction's normal & reverse value).
      • flex-end: This will place the flex-items at the end of the flex-direction of the main axis (relative to flex-direction's normal & reverse value).
      • start: This will place the flex-items at the start of the main axis. Left for flex-direction: row or row-reverse & top for flex-direction: column or column-reverse.
      • end: This will place the flex-items at the end of the main axis. Right for flex-direction: row or row-reverse & bottom for flex-direction: column or column-reverse.
      • center: This will places the flex items at the center of the flex container's main axis.
      • space-between: This will place the first flex item at the start of the main axis & the last flex item at the end of the main axis. Then space on the main axis is distributed equally among the the flex items.
      • space-around: This will distribute space equally between the flex items. The space before the first and after the last item equals half of the space between each pair of adjacent items.
      • space-evenly: Flex items are evenly distributed with equal space around them.
      • stretch, normal
    • Syntax: 
          
        .container {
            display: flex;
            justify-content: center;
        }
    • Demo: Go


  9. align-items property:(flex-container property)

    • The align-items property handles the alignment of flex items on the cross-axis of the flex container.
    • Values can be:
      • streach: If the items have with or height set to "auto" along cross axis then, this value stretches the flex items to fill up the space within the flex-container. (default value)
      • flex-start: This will place the flex items at the start of the cross-axis of the flex container. If the cross-axis is vertical then it will place the items at the top.
      • flex-end: This will place the flex items at the end of the cross-axis of the flex container. If the cross-axis is vertical then it will place the items at the bottom.
      • center: This will align flex items at the center of the cross-axis of the flex container.
      • baseline
    • Syntax: 
          
        .container {
            display: flex;
            align-items: center;
        }
    • Demo: Go


  10. align-content property:(flex-container property)

    • The align-content CSS property sets the distribution of space between and around flex lines along cross axis.
    • This property works only when flex-wrap is set to wrap or wrap-reverse and the flex items are wrapped into multiple lines or columns.
    • Values can be:
      • normal or stretch:(Default value) This will stretch the flex lines to take up the remaining space along cross axis. We can see the effect only when the width or height along cross axis is set to auto.
      • flex-start: This will place the flex lines at the start of the container's cross-axis. For example, if the cross axis is vertical then it will place the flex lines at the top.
      • flex-end: This will place the flex lines at the end of the container's cross-axis.
      • center: This will place the flex lines at the center of the container's cross-axis.
      • space-between: This will place the first flex line at the start of the cross-axis & the last flex line at the end of the cross axis. Then space on the cross-axis is distributed equally between the the lines.
      • space-around: This will distribute space equally between the flex lines. The space before the first and after the last flex line equals half of the space between each pair of adjacent flex lines.
      • space-evenly: Flex lines are evenly distributed with equal space around them.
    • Syntax: 
          
        .container {
            display: flex;
            flex-wrap: wrap;
            align-content: center; 
        }
    • Demo: Go


  11. order property:(flex-item property)

    • The order CSS property set the ordering of the flex items.
    • The items are arranged in the ascending order according to their order number value.
    • Default value 0.
    • Practice using the order property: Even though the order value change the appearance on screen, the order in the HTML remains unchanged. And a screen reader software will read according to the HTML element ordering.
    • Syntax: 
          
        .item1 {
            order: 1; 
        }
    • Demo: Go


  12. align-self property:(flex-item property)

    • The align-self CSS property works the same as align-items property but applied on individual flex items.
    • The align-self property overrides the align-items property.
    • Values are also same as align-items:
      • auto: (Default value) The flex item inherits its parent container's align-items property value.
      • streach
      • flex-start
      • flex-end
      • center
      • baseline
    • Syntax: 
          
        .item1 {
            align-self: center;
        }
    • Demo: Go


  13. flex-grow property:(flex-item property)

    • The flex-grow CSS property defines the ability for a flex item to grow if there is extra space along main axis.
    • By default flex-grow value is zero(0), means "off". Greater then 0 means "on".
    • Value: Unit less non-negative number.(can be a point number).
    • The browser treats the extra space as a value of 1. This means when we give a flex-grow value of 0.5 to only one of the flex items, the browser will add half of the remaining space to the item's size.
    • If we set a flex-grow value of 1 to only one of the flex items, the browser will add all the extra space to that item.
    • If we set a flex-grow value to only one of the flex items, then there is no difference between value 1 & 999 (any number greater then 1).
    • If we set a flex-grow value to multiple flex items, then the space is distribute among these flex items according to the ratio of their flex-grow value.
    • We can limit the maximum growth of an flex item by setting a max-width value.
    • Syntax: 
          
        .item1 {
            flex-grow: 1;
        }
    • Demo: Go


  14. flex-shrink property:(flex-item property)

    • Default value of flex-shrink is 1, means "on". We can off the flex item shrinking property by setting the value "0".
    • If the combined width of the items along the main axis is greater then the width of the flex container then they are shrinked down to fit into the container.
    • Value: Unit less non-negative number.
    • If we set different flex-shrink value to different flex items, then the shrinking happen according to the ratio of their value.
    • We can limit the maximum shrinking of an flex item by setting a min-width value.
    • Syntax: 
          
        .item1 {
            flex-shrink: 2;
        }
    • Demo: Go


  15. flex-basis property:(flex-item property)

    • The flex-basis CSS property sets the initial main size of a flex item.
    • Default value "auto".
    • flex-basis has more priority then width & height.
    • Syntax: 
          
        .item1 {
            flex-basis: 100px;
        }
    • Demo: Go


  16. flex property:(flex-item property)

    • The flex CSS property is a shorthand property for:
      • flex-grow
      • flex-shrink
      • flex-basis
    • Default values for flex is "1 0 auto". Values are seperated by a blank space.
    • 1 value: "flex-grow" (if unitless) or "flex-basis" (with unit)
    • 2 value: "flex-grow | flex-shrink (if unitless)" or "flex-grow | flex-basis (with unit)"
    • 3 value: "flex-grow | flex-shrink | flex-basis"
    • Syntax: 
          
        .item1 {
            flex: 1 1 100px;
        }
    • Demo: Go
