Chapter 7: Background & Filter

  8. filter property

Explantion of the topics:

  1. background-clip property:

    • The background-clip CSS property sets whether an element's background (color or image) extends underneath its border box, padding box, or content box.
    • The values can be:
      • border-box: (Default value) The background extends to the outside edge of the border.
      • padding-box: The background extends to the outside edge of the padding.
      • content-box: The background is painted within the content box.
      • text: The background is painted within the foreground text. If the text color is transparent then then the background will be visible.
    • Example: Go

  2. background-image property:

    • The background-image property sets one or more background images or gradient for an element. Default value is "none".
    • Syntax for image: background-image: url("file location path or web link");.
    • Gradient values will be discussed in the gradient section of this page.
    • This property can have multiple comma sepetated values. Example: background-image: url(1), url(2), url(3);, in this example the url(1) image will be in the front and url(3) image will be in the back.
    • If the image is larger then the container area then only a portion of that image will cover up the whole background.
    • On the other hand if the image is smaller then the container area then image is repeated multiple times like tiles to fill the whole background. To prevent this repeatation we can use background-repeat property.

  3. background-repeat property:

    • The background-repeat property sets if/how a background image will be repeated.
    • The values can be:
      • repeat: (Default value) The background image is repeated both vertically and horizontally. The last image will be clipped if it does not fit.
      • no-repeat: The background-image is not repeated. The image will only be shown once.
      • repeat-x: The background image is repeated only horizontally.
      • repeat-y: The background image is repeated only vertically.
    • Other values are space, round.
    • This property can also have two space separated values for horizontal & vertical.

  4. background-size property:

    • The background-size property specifies the size of the background images.
    • The values can be:
      • auto: (Default value) The background image is displayed in its original size.
      • contain: Resize the background image to make sure the image is fully visible without cropping or streaching. Rest of the unfilled area of the background is covered by repeat of that image.
      • cover: Resize(while preserving its ratio) the background image to cover the entire container. If the proportions (width:height) of the background differ from the element, the image is cropped either vertically or horizontally.
    • Other values are length values(Example: 200px;) & percentage value(Example: 33%;).
    • This property can also have two space separated values (first value: width of the image, second value: height).

  5. background-position property:

    • The background-position property sets the starting position of a background image.
    • By default(value left top or 0% 0%.), a background-image is placed at the top-left corner of an element.
    • This property can have 1, 2, 3 or 4 values sepetated by space. 1 & 2 value syntax is enough for most of the case.
    • The values can be:
      • Keywords: top, bottom, left, right, center. If we only specify one keyword, the other value will be "center".
      • Percentage (x% y%): If we only specify one value, the other value will be 50%.
      • Length (xpos ypos): If we only specify one value, the other value will be 50%.

  6. Gradient value:

    • There are different types of gradient values which are set on the background-image property to make differ types of gradient for background.
    • Different types of gradient values are:
      • linear-gradient() (Mostly used gradient value). Example: linear-gradient(red, green, ...), linear-gradient(45deg, red, green, blue ...).
      • radial-gradient()
      • conic-gradient()
      • repeating-linear-gradient()
      • repeating-radial-gradient()
      • repeating-conic-gradient()

  7. background property:

    • The background shorthand CSS property sets all background style properties at once, such as color, image, origin and size, or repeat method.
    • No need to learn or memorize this property (very complex syntax), just see the web documentation like "MDN docs" when needed.
    • This property is a shorthand for the following CSS properties:
      • background-color
      • background-clip
      • background-image
      • background-size
      • background-repeat
      • background-position
      • background-attachment
      • background-origin

  8. filter property:

    • The filter CSS property applies graphical effects like blur or color shift to any html element. Filters are commonly used to adjust the rendering of images, backgrounds, and borders.
    • filter proper value: none | blur() | brightness() | contrast() | drop-shadow() | grayscale() | hue-rotate() | invert() | opacity() | saturate() | sepia() | url();
    • MDN documentation for different types of filter values.
    • To use multiple filters, separate each filter with a space. Example: filter: contrast(200%) brightness(150%);
