Chapter 2: Color in CSS



Topics we are going to learn in this page:

  1. Color Value
  2. Color property
  3. background-color property
  4. Opacity property


Explantion of the topics:

  1. Color Values:

    • Color values are specified using predefined color names, or RGB, HSL, hexadecimal, RGBA, HSLA, HWB values.
      1. Named color: We can use name of a color like red, blue, black, or lightseagreen as color value.
      2. RGB color:We can specify colour as red, green, blue value inside rgb() function. Inside the rgb() function we can set the rgb values as 0 to 255 number or as 0% to 100%. Example: rgb(255, 99, 71) .
      3. RGBA color:RGBA color values are an extension of RGB color values with an alpha channel - which specifies the opacity or transparency for a color. The transparency value is 0 to 1 or 0% to 100%. Example: rgba(255, 84, 13, 0.7)
      4. Hexadecimal color: Hexadecimal color are written as #rrggbb. Where rr (red), gg (green) and bb (blue) are hexadecimal integers between 00 and ff, specifying the intensity of the color. Example: #6a5acd. We can also use 3-digit hex code(#RGB) when both the values (RR, GG, and BB) are the same for each component. So, if we have #ff00cc, it can be written like this: #f0c. We can also use #RRGGBBAA & #RGBA, where A & AA is the value of alpha channel (opacity or transparency).
      5. HSL color: In CSS, a color can be specified using hue, saturation, and lightness (HSL) in the form: Hue value is a degree on the color wheel from 0 to 360, saturation & lightness values are from 0% to 100%. Example: hsl(172, 90%, 50%)
      6. HSLA color:HSLA color values are an extension of HSL color values, with an Alpha channel - which specifies the opacity for a color or transparency. Example: hsla(224, 85%, 41%, 0.4)

  2. color property

    • The color property specifies the color of the text inside the element.

  3. background-color property

    • The background-color CSS property sets the background color of an element.

  4. opacity property

    • The opacity CSS property sets the opacity of an element. Opacity is like the alpha channel of color but this value apply the transparency on everything inside the element (including color, text, image). Opacity value can be 0 to 1 or 0% to 100%.
