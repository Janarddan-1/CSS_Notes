Explantion of the topics:

  1. font-family property:

    • The font-family property specifies the font for an element.
    • The font-family property can hold several font names as a "fallback" system. If the browser does not support the first font, it tries the next font.
    • Start with the font you want, and always end with a generic family, to let the browser pick a similar font in the generic family, if no other fonts are available.


  2. Adding custom font

    • We can add custom font in our website, which fonts are not present in the browser or in the OS.
    • Here is the step by step process to add custom font using google font:
      1. Visit: fonts.google.com
      2. Search and select the font
      3. Click on "Get font" or Cart symbol on the top right corner of the page
      4. Click on "Get embed code"
      5. Copy the <link> tag code & paste it inside the head part of the HIML file.
        or,
        Copy the @import code & paste it inside <style> tag of HTML or inside CSS file.
      6. Now we can use this custom font in out site by adding the font name in the font-family css property.

  3. font-size property:

    • The font-size property sets the size of a font.
    • The values can be (Size units will be discussed in the Unit chapter) :
      •     
        /* absolute-size values */
        font-size: xx-small;
        font-size: x-small;
        font-size: small;
        font-size: medium;
        font-size: large;
        font-size: x-large;
        font-size: xx-large;
        font-size: xxx-large;
        
        /* relative-size values */
        font-size: smaller;
        font-size: larger;
        
        /* length values */
        font-size: 12px;
        font-size: 0.8em;
        
        /* percentage values */
        font-size: 80%;
        
        /* math value */
        font-size: math;
        
        /* Global values */
        font-size: inherit;
        font-size: initial;
        font-size: revert;
        font-size: revert-layer;
        font-size: unset;

  4. font-weight property:

    • The font-weight property sets how thick or thin characters in text should be displayed.
    • The values can be:
      •     
        /* font-weight-absolute keyword values */
        font-weight: normal;
        font-weight: bold;
        
        /* font-weight-absolute numeric values [1,1000] */
        font-weight: 100;
        font-weight: 200;
        font-weight: 300;
        font-weight: 400; /* normal */
        font-weight: 500;
        font-weight: 600;
        font-weight: 700; /* bold */
        font-weight: 800;
        font-weight: 900;
        
        /* Keyword values relative to the parent */
        font-weight: lighter;
        font-weight: bolder;
        
        /* Global values */
        font-weight: inherit;
        font-weight: initial;
        font-weight: revert;
        font-weight: revert-layer;
        font-weight: unset;

  5. font-style property:

    • The font-style CSS property sets whether a font should be styled with a normal, italic, or oblique face from its font-family.
    • The values can be:
      •     
        font-style: normal;
        font-style: italic;
        font-style: oblique;
        font-style: oblique 10deg;

        /* Global values */
        font-style: inherit;
        font-style: initial;
        font-style: revert;
        font-style: revert-layer;
        font-style: unset;

  6. font property

    • The font CSS shorthand property sets all the different properties of an element's font.
    • This property is a shorthand for the following CSS properties: font-family, font-size, font-stretch, font-style, font-variant, font-weight, line-height
    • It is better to use font properties separately because it's complexity of specific ordering.


  7. text-align property:

    • The text-align property specifies the horizontal alignment of text in an element.
    • The values can be:
      •     
        /* Keyword values */
        text-align: start;
        text-align: end;
        text-align: left;
        text-align: right;
        text-align: center;
        text-align: justify;
        text-align: justify-all;
        text-align: match-parent;

        /* Block alignment values (Non-standard syntax) */
        text-align: -moz-center;
        text-align: -webkit-center;

        /* Global values */
        text-align: inherit;
        text-align: initial;
        text-align: revert;
        text-align: revert-layer;
        text-align: unset;

  8. letter-spacing property:

    • The letter-spacing property increases or decreases the space between characters in a text. The value can be negative.
    • The values can be:
      •     
        /* Keyword value */
        letter-spacing: normal;

        /* length values */
        letter-spacing: 0.3em;
        letter-spacing: 3px;
        letter-spacing: 0.3px;

        /* Global values */
        letter-spacing: inherit;
        letter-spacing: initial;
        letter-spacing: revert;
        letter-spacing: revert-layer;
        letter-spacing: unset;

  9. word-spacing property

    • The word-spacing property increases or decreases the white space between words. Similar to letter-spacing the value can be negative.
    • The values can be:
      •     
        /* Keyword value */
        word-spacing: normal;

        /* length values */
        word-spacing: 3px;
        word-spacing: 0.3em;

        /* Global values */
        word-spacing: inherit;
        word-spacing: initial;
        word-spacing: revert;
        word-spacing: revert-layer;
        word-spacing: unset;

  10. line-height property:

    • The line-height property specifies the height of a line.
    • The values can be a keyword "normal", non-negative numeric value with or without unit, percentage value and other global values (inherit, initial, revert, revert-layer, unset).
    • The numeric value without unit represent the height of the line as the number multiplied by the font size. Percentage value also works the same.


  11. text-shadow property:

    • The text-shadow property adds shadow to text.
    • We can add multiple shadow to a text by giving multiple sets of shadow values seperated by comma. Eg: text-shadow: 2px 2px #f00, 4px 4px 3px #0f0;
    • Each shadow is specified as two compulsory values offset-x and offset-y, followed by two optional value blur-radius & color. Example:
      •     
        /* offset-x | offset-y | blur-radius | color */
        text-shadow: 1px 1px 2px black;

        /* color | offset-x | offset-y | blur-radius */
        text-shadow: #fc0 1px 0 10px;

        /* offset-x | offset-y | color */
        text-shadow: 5px 5px #558abb;

        /* color | offset-x | offset-y */
        text-shadow: white 2px 5px;

        /* offset-x | offset-y
        /* Use defaults for color and blur-radius */
        text-shadow: 5px 10px;

        /* Global values */
        text-shadow: inherit;
        text-shadow: initial;
        text-shadow: revert;
        text-shadow: revert-layer;
        text-shadow: unset;

  12. text-transform property:

    • The text-transform property is used to turn the text into all-uppercase or all-lowercase letters, or capitalize the first letter of each word.
    • The values can be:
      •     
        /* Keyword values */
        text-transform: none;
        text-transform: capitalize;
        text-transform: uppercase;
        text-transform: lowercase;
        text-transform: full-width;
        text-transform: full-size-kana;

        /* Global values */
        text-transform: inherit;
        text-transform: initial;
        text-transform: revert;
        text-transform: revert-layer;
        text-transform: unset;

  13. text-decoration property:

    • The text-decoration property is a shorthand property for text-decoration-line (required), text-decoration-color (optional), text-decoration-style (optional), text-decoration-thickness (optional)
    • This property is used to add a decoration line to text.
    • We can add multiple text-decoration-line value (no comma is needed in between.)

    • text-decoration-line property sets the kind of line decoration that is used on text in an element, such as an underline or overline


    • text-decoration-color property sets the color of decorations added to text by text-decoration-line.


    • text-decoration-style property sets the style of the lines specified by text-decoration-line. The style applies to all lines that are set with text-decoration-line.


    • text-decoration-thickness property is used to set the thickness of the decoration line.

  14. list-style-type property:

    • The list-style-type property sets the marker (such as a disc, character, or custom counter style) of a list item element.
      •     
        /* Partial list of types */
        list-style-type: disc;
        list-style-type: circle;
        list-style-type: square;
        list-style-type: decimal;
        list-style-type: georgian;
        list-style-type: trad-chinese-informal;
        list-style-type: kannada;

        /* string value */
        list-style-type: "-";

        /* Identifier matching an @counter-style rule */
        list-style-type: custom-counter-style;

        /* Keyword value */
        list-style-type: none;

        /* Global values */
        list-style-type: inherit;
        list-style-type: initial;
        list-style-type: revert;
        list-style-type: revert-layer;
        list-style-type: unset;

  15. Pseudo-classes:

    • A pseudo-class is used to define a special state of an element.
    • For example, it can be used to: Style an element when a user mouses over it (:hover), Style visited (:visited) and unvisited links differently, Style an element when it gets focus

    • :visited works like a special selector. This pseudo-class applies only to <a> and <areagt elements that have an href attribute. This apply the css properties applies once the link has been visited by the user. Example: a:visited{css properties}.


    • :hover works like a special selector. It is generally apply the css properties when the user hovers over an element with the cursor (mouse pointer). Example: button:hover{css properties}.

  16. cursor
