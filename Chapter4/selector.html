Explantion of the topics:

  1. Universal selectors:

    • The universal selector (*) select all HTML elements on the page. C an also select all elements inside another element.
    • Syntax:
      •     
        * {
            css declarations;
        }
    • Example: Go

  2. Element Selectors:

    • The element selector selects all elements with the specified element name (Element name means tag name of html element).
    • Syntax:
      •     
        tag_name {
            css declarations;
        }
    • Example: Go

      HTML:

          
        <h1>Element Selector Example:</h1>
        <p>Paragraph 1</p>

        <h2>H2 Element</h2>
        <p>Paragraph 2</p>
        <p>Paragraph 3</p>

      CSS:

          
        p{
            color:red;
        }

      This will select all the <p> tag elements & make the text color red.

  3. ID Selectors:

    • The CSS ID selector selects an element based on the value of the element's id attribute.
    • According to html convention id name should only be used by one HTML element within a page.
    • In CSS id valued elements are accessed by a "hash" symbol (#).
    • Syntax:
      •     
        #id_name {
            css declarations;
        }
    • Example: Go

      HTML:

          
        <p>Text 1</p>
        <p id="paragraph2">Text 2</p>
        <p>Text 3</p>
        <p>Text 4</p>

      CSS:

          
        #paragraph2{
            color:#f69;
        }

      This will select the only <p> tag elements with id name "paragraph2" & apply the specific color to the text.

  4. Class Selectors:

    • The class selector selects all the elements with a specific class attribute.
    • Unlike ID, we can apply same class name to multiple HTML elements in a page.
    • In HTML if an element's class attribute value has a space in between that means that element has multiple class name. Eg: <p class="Red Text">, means this <p> tag has two class name "Red" & "Text".
    • In CSS class valued elements are accessed by a "dot"(.) symbol.
    • Syntax:
      •     
        .class_name {
            css declarations;
        }
    • Example: Go

      HTML:

          
        <ol>
            <li class="good">List Item 1</li>
            <li class="good">List Item 2</li>
            <li class="good">List Item 3</li>
            <li>List Item 4</li>
            <li>List Item 5</li>
        </ol>

      CSS:

          
        .good{
            font-weight: bold;
            color: #0A4;
        }

      This will select all the <p> tag elements with class name "good" & apply the css properties.

  5. Attribute selectors:

    • The CSS attribute selector matches elements based on the element having a given attribute explicitly set, with options for defining an attribute value or substring value match.
    • Syntax (There are multiple types of syntax):

      1. [attribute_name] : Selects all elements with an attribute name "attribute_name".

      2. tag_name[attribute_name] : Selects all "tag_name" elements with an attribute name "attribute_name".

      3. tag_name[attribute_name="val"] : Selects all "tag_name" elements with a "attribute_name" attribute whose value is exactly "val".

      4. tag_name[attribute_name*="val"] : Selects all "tag_name" elements with a "attribute_name" attribute value containing the substring "val".

      5. [attribute_name^="val"] : Selects all elements with a "attribute_name" attribute whose value starting with "val".

      6. [attribute_name$="val"] : Selects all elements with a "attribute_name" attribute whose value ending with "val".

      7. [attribute_name~="val"] : Selects all elements with a attribute "attribute_name" that contains a space-separated list of words, one of which is "val".

    • Example: Go

      HTML:

          
        <h1 title="section">Attribute selector example:</h1>

        <ul title="section">
            <li><a href="#internal">Internal link</a></li>
            <li><a href="http://example.com">Example link</a></li>
            <li><a href="#InSensitive">Insensitive internal link</a></li>
            <li><a href="http://example.org">Example org link</a></li>
            <li><a href="https://example.org">Example https org link</a></li>
        </ul>

        <form title="section">
            <input type="text" placeholder="Name"><br>
            <button title="click_to_submit_the_form">Submit</button>
        </form>

      CSS:

          
        [title]{
            border: 2px solid blue;
            width: 400px;
        }
        
        input[type="text"]{
            background-color: #6969;
        }
        
        [title*="submit"]{
            width: 200px;
            background-color: gold;
        }
        
        a[href^="#"] {
            color: #690;
        }
        
        a[href$=".org"] {
            color: red;
        }

  6. Grouping of selectors:

    • The CSS grouping selector is used to select multiple elements and style them together.
    • This reduces the code and extra effort to declare common styles for each element. To group selectors, each selector is separated by a space.
    • Different elements are separated by comma(,) symbol.
    • Syntax:
      •     
        element_1, element_2, element_3 {
            css declarations;
        }
    • Example: Go

      HTML:

          
        <h1>Heading 1</h1>
        <a href="https://www.google.com/">Google</a>
        <h2>Heading 2</h2>  
        <p>This is a paragraph.</p>

      CSS:

          
        h1, h2, p{
            color: orange;
        }

      This css code will select <h1>, <h2> & <p> tag elements together & apply the css properties.

  7. Descendant Combinator selectors:

    • The descendant combinator selector matches all elements that are descendants (direct child, grand child, grand grand child & so on) of a specified element.
    • Parent and descendant elements are seperated by a blank space.
    • Syntax:
      •     
        parent_selector descendant_selector {    
            css declarations;      
        }
    • Example: Go

      HTML:

          
        <p>Paragraph 1 (Outer Div)</p> 
    
        <div> 
            <p>Paragraph 2 (Inner Div)</p>         
            <ul>
                <li>
                    <p>Paragraph 3 (Inner list)</p>
                </li>
            </ul>
            <p>Paragraph 4 (Inner Div)</p> 
            <p>Paragraph 5 (Inner Div)</p>
        </div>

      CSS:

          
        div p{
            background-color:  lightblue;
        }

      The example selects all <p> elements inside <div> elements & apply the css property.

  8. Child Combinator selectors:

    • The child combinator selector only selects all elements that are direct child of a specified element.
    • Parent and descendant elements are seperated by a greater then (>) symbol.
    • Child Combinator Syntax:
      •     
        parent_selector > descendant_selector {    
            css declarations;      
        }
    • Example: Go

      HTML:

          
        <p>Paragraph 1 (Outer Div)</p> 
    
        <div> 
            <p>Paragraph 2 (Inner Div)</p>       
            <ul>
                <li>
                    <p>Paragraph 3 (Inner list)</p>
                </li>
            </ul>
            <p>Paragraph 4 (Inner Div)</p> 
            <p>Paragraph 5 (Inner Div)</p>
        </div>

      CSS:

          
        div > p{
            background-color:  #f696;
        }

      The example selects only the <p> elements which are direct child of <div> elements & apply the css property.

  9. Adjacent Sibling Combinator selectors:

    • The adjacent sibling selector is used to select an element that is directly after another specific element and both are children of the same parent element.
    • Both sibling elements are seperated by a plus (+) symbol.
    • Adjacent Sibling Combinator Syntax:
      •     
        sibling_selector + sibling_selector {    
            css declarations;      
        }
    • Example: Go

      HTML:

          
        <div class="red"></div>
        <div class="green"></div>
        <div class="green"></div>
        <div class="blue"></div>
        <div class="green"></div>
        <div class="red"></div>
        <div class="blue"></div>
        <div class="green"></div>

        <!-- CSS -->
        <style>
            div{
                width: 150px;
                height: 20px;
                margin: 5px 0;
            }
            .red{
                background-color: #f009;        
            }
            .green{
                background-color: #0f09;    
            }
            .blue{
                background-color: #00f9;    
            }
        </style>

      CSS:

          
        .red + .green{
            width: 300px;
        }

      The example selects the elements with class "green" only when "green" class element comes immidiately after a "red" class element & apply the css property.

  10. Subsequent Sibling Combinator selectors:

    • The subsequent sibling combinator selector selects all elements that are next siblings of a specified element. and both are children of the same parent element.
    • Both sibling elements are seperated by a tilde (~) symbol.
    • Subsequent Sibling Combinator Syntax:
      •     
        sibling_selector ~ sibling_selector {    
            css declarations;      
        }
    • Example: Go

      HTML:

          
        <div class="red"></div>
        <div class="blue"></div>
        <div class="green"></div>
        <div class="green"></div>
        <section>
            <div class="blue"></div>
            <div class="green"></div>
            <div class="red"></div>
        </section>
        <div class="blue"></div>
        <div class="green"></div>


        <!-- CSS -->
        <style>
            div{
                width: 150px;
                height: 20px;
                margin: 5px 0;
            }
            .red{
                background-color: #f009;        
            }
            .green{
                background-color: #0f09;    
            }
            .blue{
                background-color: #00f9;    
            }
        </style>

      CSS:

          
        .red ~ .green{
            width: 300px;
        }

      The example selects all the elements with class "green" when "green" class element is a sibling (same parent) of "red" class element & apply the css property.

  11. Compound selectors:

    • A compound selector is a sequence of simple selectors that are not separated by a combinator.
    • A compound selector represents a set of simultaneous conditions on a single element. A given element is said to match a compound selector when the element matches all the simple selectors in the compound selector.
    • The conditions can be specific class name or id or any other attribute. There is no white-space or any symbol between those simple selector.
    • Syntax:
      •     
        SelectorCondition {
            css declarations;
        }
    • Example: Go

      HTML:

          
        <div>
            <p>paragraph 1</p>
            <ul>
                <li class="notes">List item 1</li>
                <li title="wiki">List item 2</li>
                <li title="web"><a href="#">Link 1</a></li>
                <li class="notes"><a href="#">Link 2</a></li>
                <li title="web"><a href="#">Link 3</a></li>
            </ul>
        </div>
        <div class="notes">Text 1</div>
        <div class="notes">Text 3</div>

      CSS:

          
        div.notes{
            background-color: aqua;
        }
        
        li[title]{
            font-weight: bold;
        }

      This will select only those <div> tag elements which have a class name "notes" & make the background color aqua. And also select only those <li> tags which have a title attribute & make the text bold.

  12. Pseudo class selectors:

    • A pseudo-class is used to select a special state of an element.
    • For example, it can be used to Style an element when a user mouses over it (:hover), Style visited (:visited) and unvisited links differently, Style an element when it gets focus (:focus )
    • :nth-of-type() matches elements based on their position among siblings of the same type (tag name).
    • The :first-child CSS pseudo-class represents the first element among a group of sibling elements.
    • Syntax:
      •     
        Selector:pseudo-class {
            css declarations;
        }
    • Example: Go

      HTML:

          
        <a href="https://www.google.com/">Google</a>
           
        <a href="https://www.wikipedia.org/">Wikipedia</a><br>
        <a href="https://www.w3schools.com/cssref/css_selectors.php">CSS Selector Reference (w3schools)</a><br><br><br>

        <form action="">
            <input type="text" name="username" placeholder="Enter your username" /><br>
            <input type="email" name="emailid" placeholder="Enter your email-id" /><br>
            <button>Submit</button>
        </form>
        <ol>
            <li>Item 1</li>
            <li>Item 2</li>
            <li>Item 3</li>
            <li>Item 4</li>
            <li>Item 5</li>
            <li>Item 6</li>
            <li>Item 7</li>
        </ol>

      CSS:

          
        a:visited {
            color: forestgreen;
        }
        /* This :visited pseudo class affect is not showing properly in codepen.io */
        /* codepen.io marking all the link as visited. */
        input:focus{
            background-color: #ff06;
        }
        button:hover{
            background-color: #0ff6;
        }
        li:nth-of-type(2n-1){
            background-color: #0f06;
        }
        li:nth-child(2){
            color:#e95400;
        }

      This above css will select the input box and app the css property when we click on them. (For Submit button) Apply the css properties when we hover over the submit button. (For links) Apply the css properties when we clinked on the links.

  13. Pseudo Element selectors:

    • A CSS pseudo-element is used to style specified parts of an element.
    • For example, Style the first letter (::first-letter), or first line (::first-line ) of an element
    • Syntax:
      •     
        Selector::pseudo-element {
            css declarations;
        }
    • Example: Go

      HTML:

          
        <div>
            <h2>Pseudo Element selectors:</h2>

            <p>Scientists exploring the depths of Monterey Bay unexpectedly encountered a rare and unique species of dragonfish. This species is the rarest of its species.</p>

            <p>When Robison and a team of researchers discovered this fish, they were aboard a week-long expedition.</p>
        </div>

      CSS:

          
        p{
            font-family: open sans;
            text-align: justify;
        }
        p::first-letter {
            font-size: 18px;
            font-weight: bold;
            color: #e95400;
        }
        div{
            width: 270px;
            border: 1px solid #00f;
            padding: 0 32px;
            background-color: #f0f1;
            margin: auto;
        }

      This above css will select first letter of all the <p> tag element and apply the css styling.

  14. Specificity:

    • If there are two or more CSS selector point to the same element, the selector with the highest specificity value will "win".
    • The specificity value decreases in the following top to bottom order:
      • Inline style
      • ID selectors
      • class, attributes & pseudo-classes selectors
      • Element & pseudo-element selectors
    • We can calculate the exact specificity value of a selector using 3 comma separated value(x,y,z).
      • 1st value is the total number of id selectors,
      • 2nd value is the total number of class, attributes or pseudo-classes selectors,
      • 3rd value is the total number of element & pseudo-element selectors
      • The in between space and any other symbols('+', '~', ',') have no value or significance in specificity calculation.
      • In vs-code if we hover over a css selector in a css file then the specificity value is shown as a popup.
      • Example: 
            
        div p {                     /* Specificity= (0,0,2) */
            
        } 

        div>p>#green {              /* Specificity= (1,0,2) */
            
        } 

        .top span{                  /* Specificity= (0,1,1) */
            
        } 

        .main .text .wide {         /* Specificity= (0,3,0) */
            
        } 

        div p>#green ~ a:hover {    /* Specificity= (1,1,3) */
            
        }
    • Comparing specificity values:
      • First compare the 1st value of different selector, the selector with a greater number is selected & if both number are same then compare the 2nd value & similarly if 2nd vlaue is also same then compare between 3rd value.
      • If 1st, 2nd & 3rd all values are same then the selector which is defined later is selected.
