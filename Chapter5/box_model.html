  1. border Property:

    • The CSS border properties used to specify the style, width, and color of an element's border.
    • It is a shorthand property for border-width, border-style (required), border-color
    • border-width:
      1. This property value can be thin, medium, thick and any size value. Can set 1 (same in all 4 sides) or 2(top & bottom | left & right) or 3(top | left & right | bottom ) or 4(top | right | bottom | left) values.
      2. It is a shorthand property for border-top-width, border-right-width, border-bottom-width, border-left-width. We can set this properties separately to use as a alternative of underline.
    • border-style:
      1. This property value can be none, hidden, dotted, dashed, solid, double, groove, ridge, inset, outset.. Can set 1 (same in all 4 sides) or 2(top & bottom | left & right) or 3(top | left & right | bottom ) or 4(top | right | bottom | left) values.
      2. It is a shorthand property for border-top-style, border-right-style, border-bottom-style, border-left-style.
    • border-color:
      1. We can set 1 (same in all 4 sides) or 2(top & bottom | left & right) or 3(top | left & right | bottom ) or 4(top | right | bottom | left) color values.
      2. It is a shorthand property for border-top-color, border-right-color, border-bottom-color, border-left-color.
    • The border shorthand property values can be:
      •     
        /* style */
        border: solid;

        /* width | style */
        border: 2px dotted;

        /* style | color */
        border: outset #f33;

        /* width | style | color */
        border: medium dashed green;

        /* Global values */
        border: inherit;
        border: initial;
        border: revert;
        border: revert-layer;
        border: unset;

  2. width & heignt Property:

    • The width & height CSS property sets an element's width & height respectively.
    • By default, these properties set the width & height of the content area (without including the padding & border width).
    • But if box-sizing is set to border-box; then these properties set the width & height of t he border area (including the paddin & border width).
    • If we set the value as a percentage(%) value then the width & height are set according to the width & height of it's parent container.
    • The width property value can be:
      •     
        /* length values */
        width: 300px;
        width: 25em;
        width: anchor-size(--myAnchor inline, 120%);
        width: minmax(100px, anchor-size(width));
        
        /* percentage value */
        width: 75%;
        
        /* Keyword values */
        width: max-content;
        width: min-content;
        width: fit-content;
        width: fit-content(20em);
        width: auto;
        
        /* Global values */
        width: inherit;
        width: initial;
        width: revert;
        width: revert-layer;
        width: unset;
    • Example:
    • The height property value can be:
      •     
        /* length values */
        height: 120px;
        height: 10em;
        height: 100vh;
        height: anchor-size(height);
        height: anchor-size(--myAnchor self-block, 250px);
        height: clamp(200px, anchor-size(width));
        
        /* percentage value */
        height: 75%;
        
        /* Keyword values */
        height: max-content;
        height: min-content;
        height: fit-content;
        height: fit-content(20em);
        height: auto;
        height: minmax(min-content, anchor-size(width));
        
        /* Global values */
        height: inherit;
        height: initial;
        height: revert;
        height: revert-layer;
        height: unset;

  3. min-width & max-width Property:

    • The min-width CSS property sets the minimum width of an element.
    • If the content is smaller than the minimum width, the minimum width will be applied. If the content is larger than the minimum width, the min-width property has no effect.
    • The max-width property defines the maximum width of an element.
    • If the content is larger than the maximum width, it will automatically change the height of the element. If the content is smaller than the maximum width, the max-width property has no effect.
    • Similarly min-heignt & max-heignt property also works.
    • The min-width property value can be:
      •     
        /* length value */
        min-width: 3.5em;
        min-width: anchor-size(width);
        min-width: anchor-size(--myAnchor self-inline, 200%);
        
        /* percentage value */
        min-width: 10%;
        
        /* Keyword values */
        min-width: max-content;
        min-width: min-content;
        min-width: fit-content;
        min-width: fit-content(20em);
        
        /* Global values */
        min-width: inherit;
        min-width: initial;
        min-width: revert;
        min-width: revert-layer;
        min-width: unset;
    • Example:
    • The max-width property value can be:
      •     
        /* length value */
        max-width: 3.5em;
        max-width: anchor-size(--myAnchor inline, 245px);
        max-width: calc(anchor-size(width) + 4em);
        
        /* percentage value */
        max-width: 75%;
        
        /* Keyword values */
        max-width: none;
        max-width: max-content;
        max-width: min-content;
        max-width: fit-content;
        max-width: fit-content(20em);
        
        /* Global values */
        max-width: inherit;
        max-width: initial;
        max-width: revert;
        max-width: revert-layer;
        max-width: unset;

  4. padding Property:

    • Padding is the space optionally can be added between the outer-edge of the content-box & the inner side of the border.
    • padding is a shorthand property for padding-top, padding-right, padding-bottom, padding-left.
    • The padding property sets the padding area on all four sides of an element at once.
    • The value can be:
      •     
        /* Apply to all four sides */
        padding: 1em;

        /* top and bottom | left and right */
        padding: 5% 10%;

        /* top | left and right | bottom */
        padding: 1em 2em 2em;

        /* top | right | bottom | left */
        padding: 5px 1em 0 2em;

        /* Global values */
        padding: inherit;
        padding: initial;
        padding: revert;
        padding: revert-layer;
        padding: unset;

  5. margin Property:

    • Margins are used to create space around elements, outside of any defined borders.
    • margin is a shorthand property for margin-top, margin-right, margin-bottom, margin-left.
    • The margin sets the margin area on all four sides of an element at once.
    • Margin value can also be negative.
    • The value can be:
      •     
        /* Apply to all four sides */
        margin: 1em;
        margin: -3px;

        /* top and bottom | left and right */
        margin: 5% auto;

        /* top | left and right | bottom */
        margin: 1em auto 2em;

        /* top | right | bottom | left */
        margin: 2px 1em 0 auto;

        /* Global values */
        margin: inherit;
        margin: initial;
        margin: revert;
        margin: revert-layer;
        margin: unset;
    • Example:

  6. box-sizing Property (Alternate Box Model):

    • The box-sizing CSS property sets how the total width and height of an element is calculated.
    • Normal Box-Model: In default box model(box-sizing: content-box;), the width and height we assign to an element is applied only to the element's content box (Not including padding and border width).
    • Alternate Box-Model: If we set box-sizing: border-box;, then the width and height we assign to an element is applied to the element's border box (Including padding and border width). In this mode if we add padding and border value then the content box size become smaller.
    • The value can be:
      •     
        box-sizing: border-box;
        box-sizing: content-box;

        /* Global values */
        box-sizing: inherit;
        box-sizing: initial;
        box-sizing: revert;
        box-sizing: revert-layer;
        box-sizing: unset;
    • Example: GO

  7. display Property (inline, block, inline-block, none):

    • The display property specifies the display behavior (the type of rendering box) of an element.
    • Block-level Elements: This elements always starts on a new line, and the browsers automatically add some space (a margin) before and after the element. A block-level element always takes up the full width available (stretches out to the left and right as far as it can).Example: <h1>, <p>, <div>.
    • Inline Elements: An inline element only takes up as much width as necessary. This element does not start on a new line. Example: <a>, <span>, <input>, <img>.
    • The display property value can be block, inline, inline-block, none, flex, inline-flex, grid, inline-grid, flow-root etc.But in this chapter we will discuss only block, inline, inline-block, none.
      1. display: block; makes any element a block level element. In block-level element padding, border & margin properties are visible on all 4 sides of the element.
      2. display: inline; makes any element a inline element. In inline element padding, border & margin properties are only visible in left & right direction but not on the top & bottom direction.
      3. display: inline-block; This makes padding, border & margin effect visible on all 4 sides of the element like a block-level element. And also can have other elements on the left & right side of the element like an inline element.(Does not takes up the full width available)
      4. display: none; makes the element disappeat from the page (Not even blank space). This value is useful for responsive web-design to hide some element from the page on a smaller screen size.
    • Example: Go



  8. Margin Collapsing:

    • The top and bottom margins of blocks are sometimes combined (collapsed) into a single margin whose size is the largest of the individual margins (or just one of them, if they are equal), a behavior known as margin collapsing.
    • This does not happen on left and right margins! Only top and bottom margins! Margins don't collapse in a container with display set to flex or grid.
    • Note that the margins of floating and absolutely positioned elements never collapse.
    • When negative margins are involved, the size of the collapsed margin is the sum of the largest positive margin and the smallest (most negative) negative margin.
    • When all margins are negative, the size of the collapsed margin is the smallest (most negative) margin. This applies to both adjacent elements and nested elements.

  9. border-radius Property:

    • The border-radius CSS property rounds the corners of an element's outer border edge.
    • border-radius is a shorthand property for border-top-left-radius, border-top-right-radius, border-bottom-right-radius, border-bottom-left-radius. This individual properties can have 1 value for perfecrly rounded corner & also can have 2 space-separated values(horizontal-rdious & vertical-rdious) for elliptical corner.
    • The border-radius property value can be:
      •     
        /* without '/' */
        /* Radius is set for all 4 sides */
        border-radius: 10px;

        /* top-left-and-bottom-right | top-right-and-bottom-left */
        border-radius: 10px 5%;

        /* top-left | top-right-and-bottom-left | bottom-right */
        border-radius: 2px 4px 2px;

        /* top-left | top-right | bottom-right | bottom-left */
        border-radius: 1px 0 3px 4px;
    ------------------------------------------------------------------------------------------
        /* with '/' */
        /* horizontal radius of all 4 corner / vertical rdious radius of all 4 corner */
        border-radius: 10px / 20px;

        /* horizontal top-left-and-bottom-right | horizontal top-right-and-bottom-left / vertical top-left-and-bottom-right | horizontal top-right-and-bottom-left */
        border-radius: 10px 5% / 20px 30px;

        /* horizontal top-left | horizontal top-right-and-bottom-left | horizontal bottom-right / vertical top-left | vertical top-right-and-bottom-left | vertical bottom-right */
        border-radius: 10px 5px 2em / 20px 25px 30%;

        /* (horizontal radius values) / vertical top-left | vertical top-right | vertical bottom-right | vertical bottom-left */
        border-radius: 10px 5% / 20px 25em 30px 35em;

        /* Global values */
        border-radius: inherit;
        border-radius: initial;
        border-radius: revert;
        border-radius: revert-layer;
        border-radius: unset;

  10. box-shadow Property:

    • The box-shadow CSS property adds shadow effects around an element's frame. You can set multiple effects separated by commas.
    • Box shadow template: Go
    • Syntax:
      1. box-shadow: 10px 10px; (Only specify a horizontal and a vertical shadow. The default color of the shadow is the current text-color)
      2. box-shadow: 10px 10px lightblue; (The color parameter defines the color of the shadow)
      3. box-shadow: 10px 10px 5px lightblue; (3rd value is the blur value. The blur parameter defines the blur radius)
      4. box-shadow: 10px 10px 5px 12px lightblue; (4th value is the spread value. The spread parameter defines the spread radius. A positive value increases the size of the shadow, a negative value decreases the size of the shadow)
      5. box-shadow: 10px 10px 5px lightblue inset; (The inset parameter changes the shadow from an outer shadow (outset) to an inner shadow.)
    • Other value can be:
      •     
        /* Global values */
        box-shadow: inherit;
        box-shadow: initial;
        box-shadow: revert;
        box-shadow: revert-layer;
        box-shadow: unset;

  11. overflow Property:

    • The overflow property sets the desired behavior when content does not fit in the element's padding box (overflows) in the horizontal and/or vertical direction.
    • overflow is a shorthand property for overflow-x, overflow-y.
    • Can set 1 (same value for both overflow-x & overflow-y) or 2(overflow-x | overflow-y) values separated by blank space.
    • The overflow property has the following values:
      1. overflow: visible; (Default value. The content renders outside the element's box. The overflow is not clipped)
      2. overflow: hidden; (The overflow is clipped, and the rest of the content will be invisible)
      3. overflow: scroll; (The overflow is clipped, and a scrollbar is added to see the rest of the content)
      4. overflow: auto; (Similar to scroll, but it adds scrollbars only when necessary)
    • Other values can be:
      •     
        /* Global values */
        overflow: inherit;
        overflow: initial;
        overflow: revert;
        overflow: revert-layer;
        overflow: unset;

  12. object-fit Property:

    • The object-fit property is used to specify how an <img> or <video> should be resized to fit its container.
    • This property tells the content to fill the container in a variety of ways; such as "preserve that aspect ratio" or "stretch up and take up as much space as possible".
    • The object-fit property can take one of the following values:
      1. fill - This is default. The image is resized to fill the given dimension. If necessary, the image will be stretched or squished to fit.
      2. contain - The image keeps its aspect ratio, but is resized to fit within the given dimension.
      3. cover - The image keeps its aspect ratio and fills the given dimension. The image will be clipped to fit.
      4. none - The image is not resized.
      5. scale-down - the image is scaled down to the smallest version of none or contain.
    • Other values can be:
      •     
        /* Global values */
        object-fit: inherit;
        object-fit: initial;
        object-fit: revert;
        object-fit: revert-layer;
        object-fit: unset;
