Explantion of the topics:

  1. transition property:

    • The transitions css property is used for smooth transition of some properties of an element when we change those properties useing some pseudo class, javascript or using dev-tool.
    • This property is a shorthand property for:
      • transition-property (required)
      • transition-duration (required)
      • transition-timing-function
      • transition-delay
      • transition-behavior.
    • Syntax:

      transition: property | duration | timing-function | delay ;

    • We can assigne multiple sets of transition value seperated by comma for multiple properties. Example: transition: color 2s, background-color 3s, width 2s ease-in;
    • Or we can ues "all" keyword as property value to apply transition to all the properties of the element. Example: transition: all 1s ease-out;
    • property: This value sets the CSS properties to which a transition effect should be applied.
    • duration: This value sets the length of time a transition animation should take to complete. By default, the value is 0s, meaning that no animation will occur.
    • timing-function (optional): This value lets us establish an acceleration curve so that the speed of the transition can vary over its duration. Values can be ease, ease-in, ease-out, ease-in-out, linear etc.
    • delay: This value specifies the duration to wait before starting a property's transition effect when its value changes.(Rarely used in normal case)
    • If we mention the transition property to the normal element then the effect is shown both time when the transition starts and when ends.
    • If we mention the transition property to the element(E.g. :hover) where the property change happen then the effect is shown only when the transition starts.


  2. Best properties for transition:

    • The two most animatable properties for best performence & smoother rendering are transform & opacity.
    • Because these properties only affects the compositing part in browser rendering, does not affect the positioning of other elements.
    • Whenever we can use these 2 properties as an alternative of any other properties in a transition or animation, we should use these 2 properties.

  3. transform property:

    • The transform CSS property applies a 2D or 3D transformation to an element. This property allows you to rotate, scale, move, skew, etc., elements.
    • The values can be keyword- "none"; function value translate(x,y), rotate(angle), scale(x,y), skew(x-angle,y-angle), matrix(n,n,n,n,n,n), perspective(n) & their 3d, x, y, z variations.
    • Some of the mostly used function-values are discussed in the next section below.
    • We can assigne multiple function-values seperated by space. Example:    transform: rotate(70deg) scaleX(3);


  4. translate value (transform: translate()):

    • This value allow us to move an element laft, right, up, down or multiple at the same time.
      • translateX(length): Move the element along the X-axis.
      • translateY(length): Move the element along the Y-axis.
      • translate(X-length, Y-length): Move the element along the X & Y axis at the same time.
      • translateZ(length), translate3d(x, y, z)
    • Example: transform: translateX(400px);
      This above code will move the element 400px towards right, without affecting the sorrunding elements. And the space occupied by the element will still left behind as an empty space.
    • We should not use translate() for layout purposr(like positioning properties), theoritically we could but it is not the intended usecase. Most of the time it is used for animation purposes.

  5. rotate value (transform: rotate()):

    • This value allow us to rotate an element around a fixed point.
    • Default point on which the rotation happen is the center of the element. We can change this point using transform-origin property.
    • Variations of rotate function:
      • rotate(angle): Rotates an element around a fixed point on the 2D plane, without deforming it.
      • rotateX(angle), rotateY(angle), rotateZ(angle), rotate3d(x, y, z, angle)
    • Example: transform: rotate(45deg);
      This above code will rotate the element 45 degree clockwise.


  6. scale value (transform: scale()):

    • This value allow us to resize the element by streaching or shrinking.
    • This function takes number value or a percentage (%) value as parameter.
    • Variations of scale function:
      • scaleX(number): Increases or decreases the width of an element by streaching or shrinking horizontally.
      • scaleY(number): Increases or decreases the height of an element by streaching or shrinking vertically.
      • scale(x-number, y-number): Increases or decreases the height & width of an element by streaching or shrinking vertically & horizontally. If one parameter value is provided in the function (scale(number);) then x-scale & y-scale value will be same.
      • scaleZ(number), scale3d(x-number, y-number, z-number)
    • Example: transform: scaleX(2);
      This above code will make the width of the element double by streaching.


  7. transform-origin property:

    • The transform-origin CSS property sets the origin point for an element's transformations (rotate, scale, skew etc.). Default value center.
    • Mostly use for rotate() CSS function.
    • The values can be: 
          
        /* One-value syntax */
        transform-origin: 2px;

        /* x-offset | y-offset */
        transform-origin: 3cm 2px;

        /* x-offset-keyword | y-offset */
        transform-origin: left 2px;

        /* x-offset-keyword | y-offset-keyword */
        transform-origin: right top;

        /* y-offset-keyword | x-offset-keyword */
        transform-origin: top right;

        /* x-offset | y-offset | z-offset */
        transform-origin: 2px 30% 10px;

        /* x-offset-keyword | y-offset | z-offset */
        transform-origin: left 5px -3px;

        /* x-offset-keyword | y-offset-keyword | z-offset */
        transform-origin: right bottom 2cm;

        /* y-offset-keyword | x-offset-keyword | z-offset */
        transform-origin: bottom right 2cm;

        /* Global values */
        transform-origin: inherit;
        transform-origin: initial;
        transform-origin: revert;
        transform-origin: revert-layer;
        transform-origin: unset;
