Chapter 6: CSS Units & float



Topics we are going to learn in this page:

  1. Units in CSS
  2. Absolute units
  3. Relative units
  4. float property

Explantion of the topics:

  1. Units in CSS:

    • CSS has several different units for expressing a length. They are 2 types, absolute units & relative units.
    • No matter thatever the units are in the css file when the file is rendered on a browser, it always converted into px(absolute) unit.

  2. Absolute units

    • Absolute units specify a fixed length value. It doesn't matter if the screen's width or height changes, the value will remain fixed.
    • Some of the absolute units are px, pt, mm, cm, in.
    • px: Pixels (px) are typically the most popular absolute unit for screens.
    • pt: Point (pt) values are as the ms-word font size unit. pt, mm, cm, in units are useful when there some need for printing.

  3. Relative units

    • Relative units are useful for styling responsive sites because they scale relative to the parent or window size (depending on the unit).
    • Some of the mostly used absolute units are %, em, rem, vh, vw.
    • Percentage(%) unit: Relative to the size of the parent element.
    • em (element): Relative to the size of the parent element. Works similar like percentage(%) unit.Example: 100% = 1em, 50% = 0.5em.
    • rem (element): Relative to the size of the root element (<html> tag). It is easier to understand then em unit. By default the font size of the root element is 16px on chrome browser.
    • vh (viewport height): Relative to the height of the viewport. Viewport is the visible area of browser window. 100vh = 100% of the viewport height equivalent. This unit should only be used with a height value.
    • vw (viewport width): Relative to the width of the viewport. 100vw = 100% of the viewport width equivalent. This unit should only be used with a width value.

  4. float property:

    • The float CSS property places an element on the left or right side of its container, allowing text and inline elements to wrap around it (These other element's padding and margin do not show any affect around the float element).
    • The value of float can br none (default value), left, right, inline-start, inline-end.
